# Sekai Tools

## 主要功能

- 根据三周年后的PJSK游戏内录剧情文件和游戏内素材文件自动生成适配风格的字幕文件
    - 分辨率自适应
    - 需要视频固定帧速率
- 内嵌 Sekai Text 翻译文档编辑器
- 剧本文件下载

## 使用方法
WIP 或 加入PJS字幕组查看组内文档
